# Emoji Calculator Backend

## Installation
Step-1: <br/>
Run the command to clone the repository:
```bash
 git clone https://github.com/RakibNr30/nr-emoji-calculator-backend.git
```
Step-2:<br/>
Checkout to ``development`` branch
 ```bash
 git checkout development
 ```
Step-3:<br/>
Install the dependencies
 ```bash
 composer install
```
Step-4:<br/>
Setup and run frontend from (if not run yet)
```bash
https://github.com/RakibNr30/nr-emoji-calculator-frontend.git
```
Step-5:<br/>
Start the script by running the following command
 ```bash
 php artisan serve --port=8000
```
